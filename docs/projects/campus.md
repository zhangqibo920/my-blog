# 跨端校园服务平台

## 项目概述

面向高校师生的综合服务平台，集成课表查询、成绩查询、失物招领、二手交易等模块。

## 技术栈

| 技术 | 用途 |
|------|------|
| UniApp | 跨端开发 |
| Vue 3 | 前端框架 |
| Spring Boot | 后端框架 |
| MyBatis | ORM 框架 |
| PostgreSQL | 数据库 |

## 核心功能

- **课表查询**: 对接教务系统，实时同步课程信息
- **成绩查询**: 学期成绩、GPA 计算、成绩趋势分析
- **失物招领**: 发布、认领、地图定位
- **二手交易**: 商品发布、聊天、交易保障

## 主要工作

- 使用 UniApp 跨端开发，同时支持 Android、iOS 和微信小程序
- 设计 RESTful API 规范，编写接口文档
- 集成消息推送服务，实现校园通知实时触达
- 实现基于地理位置的附近功能

## 项目成果

- 覆盖 3 所高校，注册用户 8000+
- 日活跃用户 1200+
- 失物招领找回率 75%
