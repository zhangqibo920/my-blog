# 微信小程序商城

## 项目概述

面向 C 端用户的电商小程序，支持商品浏览、购物车、在线支付、订单管理等功能。

## 技术栈

| 技术 | 用途 |
|------|------|
| 微信小程序 | 运行时框架 |
| UniApp | 跨端开发 |
| Spring Boot | 后端框架 |
| Redis | 缓存与会话 |
| MySQL | 数据库 |

## 核心功能

- **商品展示**: 分类浏览、搜索、详情展示
- **购物车**: 加入购物车、数量修改、规格选择
- **在线支付**: 微信支付接入，支付流程闭环
- **订单管理**: 下单、支付、发货、收货全流程

## 主要工作

- 使用 UniApp 实现一套代码同时发布微信小程序和 H5
- 基于 Redis 实现购物车和库存扣减的高并发方案
- 接入微信支付 SDK，完成支付流程闭环
- 设计商品 SKU 组合方案，支持多规格商品

## 项目成果

- 小程序上线 3 个月累计用户 5000+
- 日均订单量 100+
- 支付成功率 98%
