# 企业级管理后台

## 项目概述

为公司内部打造的全栈管理后台系统，涵盖用户管理、权限控制、数据可视化等核心模块。

## 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Element Plus | UI 组件库 |
| Spring Boot | 后端框架 |
| MyBatis-Plus | ORM 框架 |
| MySQL | 数据库 |

## 核心功能

- **用户管理**: 支持多角色、多部门的用户体系
- **权限控制**: 基于 RBAC 模型的细粒度权限管理
- **数据可视化**: ECharts 构建的可视化仪表盘
- **日志审计**: 操作日志记录与审计追踪

## 主要工作

- 设计并实现了基于 RBAC 的权限管理系统，支持按钮级别的权限控制
- 使用 ECharts 构建数据可视化仪表盘，支持多维度数据筛选
- 优化首屏加载性能，LCP 从 3.2s 降至 1.1s
- 封装通用的表格、表单组件，提升开发效率

## 项目成果

- 覆盖公司 80% 的内部管理场景
- 接入 5 个业务部门，日活跃用户 200+
- 首屏加载性能提升 65%
