# 定向运动

## 项目概述

定向运动赛事管理与训练辅助平台，为定向运动爱好者和赛事组织者提供全方位的数字化支持。项目为校企合作项目，与体育机构合作开发，旨在提升定向运动赛事的组织效率和训练效果。

## 技术栈

| 技术 | 用途 |
|------|------|
| UniApp | 跨端开发 |
| Vue 3 | 前端框架 |
| Spring Boot | 后端框架 |
| PostgreSQL | 数据库 |

## 核心功能

- **赛事管理**: 赛事发布、报名、成绩公示全流程
- **路线规划**: 地图标注、路线设计、打卡点设置
- **训练记录**: 训练数据记录、成绩趋势分析
- **数据统计**: 个人成绩统计、排行榜、历史对比

## 主要工作

- 使用 UniApp 实现跨端开发，支持 Android 和微信小程序
- 集成地图服务实现路线可视化，支持地图标注、路线设计、打卡点设置
- 设计训练数据统计与分析模块，实现训练数据的可视化展示
- 实现实时成绩同步与排名计算，支持赛事成绩的实时更新
- 负责全栈开发，采用UniApp + Vue 3 + Spring Boot + PostgreSQL技术栈

## 项目成果

- 支持多场赛事的线上管理，提升赛事组织效率
- 赛事数据完善，支持多场赛事的组织和参与
- 在地图集成、数据可视化等方面有技术创新
- 项目对定向运动推广具有实际社会价值，有望在更多赛事中应用
- 积累了UniApp跨端开发、地图集成、数据可视化等方面的技术经验
