# "战书"双语阅读器

## 项目概述

双语对照阅读应用，为英语学习者提供沉浸式的中英文对照阅读体验。项目为企业项目，旨在为英语学习者提供高质量的双语阅读工具。

## 技术栈

| 技术 | 用途 |
|------|------|
| UniApp | 跨端开发 |
| Vue 3 | 前端框架 |
| Node.js | 后端框架 |
| MongoDB | 数据库 |

## 核心功能

- **双语对照**: 中英文段落逐行对照显示
- **生词标注**: 点击查词、生词本收藏、复习提醒
- **阅读进度**: 阅读时长统计、进度保存与同步
- **内容管理**: 文章分类、标签筛选、书签功能

## 主要工作

- 设计双语对照的排版与交互方案，解决中英文对齐问题，优化交互体验
- 实现生词标注与生词本功能，支持点击查词和复习提醒
- 开发阅读进度统计与数据同步，实现用户阅读时长统计
- 优化长文阅读的滚动性能，解决大量文本数据加载时的延迟问题
- 集成AI翻译接口，实现文章自动翻译，优化翻译质量、调用频率、响应时间和错误处理
- 实现跨端适配，支持微信小程序、H5、App等多个平台

## 项目成果

- 支持多部双语书籍阅读，用户阅读体验良好
- 项目在公司内部推广，获得积极反馈
- 在AI翻译集成、双语排版、跨端开发等方面有技术创新
- 积累了UniApp跨端开发、AI接口集成、性能优化等方面的技术经验
