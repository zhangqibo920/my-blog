# 大学牲 App

## 项目概述

面向大学生的校园生活服务平台，整合校园社交、跑腿、兼职、外卖、团购等常用功能，为学生提供一站式服务体验。项目为企业项目，旨在打造校园生活服务综合平台。

## 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Spring Boot | 后端框架 |
| MySQL | 数据库 |

## 核心功能

- **校园社交**: 校园圈子、动态发布、好友互动
- **跑腿服务**: 校园跑腿任务发布与接单
- **兼职平台**: 校园兼职信息发布与匹配
- **外卖团购**: 校园外卖订购与团购活动

## 主要工作

- 负责全栈开发，采用Vue 3 + Spring Boot + MySQL技术栈
- 设计并实现多模块架构，将社交、跑腿、兼职、外卖、团购等功能整合到一个平台
- 优化页面加载性能与用户体验，提升各功能模块的交互流畅度
- 实现用户认证与权限管理，确保不同功能模块的数据安全

## 项目成果

- 覆盖校内多个学院，用户反馈良好，功能持续迭代中
- 项目在多个学校推广，获得积极反馈
- 在多模块架构设计、功能整合方面有技术创新
- 积累了全栈开发、多模块架构设计、校园生活服务等方面的技术经验
