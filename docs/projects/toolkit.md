# 一起来工具箱

## 项目概述

实用工具集合平台，集成多种日常办公和开发常用的小工具，一站式解决各类需求。项目源于个人在工作学习中对各类工具的综合需求，希望通过整合提升效率。

## 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| JavaScript | 脚本语言 |

## 核心功能

- **文本工具**: JSON 格式化、Base64 编解码、URL 编解码、文本对比
- **图片工具**: 图片压缩、格式转换、尺寸调整
- **计算工具**: 单位换算、日期计算、正则测试
- **开发工具**: 代码格式化、颜色转换、正则生成器

## 主要工作

- 设计工具分类与导航架构，实现响应式布局适配不同设备
- 实现各类工具的核心算法，优化图片处理性能（内存优化、算法优化、懒加载、缓存策略）
- 优化页面加载性能，实现路由懒加载，提升首屏加载速度
- 优化工具交互体验，注重易用性和用户体验设计
- 实现数据本地存储功能，使用localStorage存储用户配置和历史记录

## 项目成果

- 集成 20+ 实用小工具，覆盖文本、图片、计算、开发等多个场景
- 轻量快速，无需注册即可使用，页面加载速度显著提升
- 用户体验良好，获得积极反馈，功能持续迭代完善
- 积累了Vue 3项目开发、性能优化、本地存储等方面的技术经验
