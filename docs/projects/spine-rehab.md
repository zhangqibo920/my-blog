# 脊柱康复

## 项目概述

脊柱康复训练管理平台，帮助康复师制定训练计划，患者按时完成康复训练并跟踪进度。项目为校企合作项目，与医疗企业合作开发，旨在提升康复训练的规范性和效率。

## 技术栈

| 技术 | 用途 |
|------|------|
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Spring Boot | 后端框架 |
| MySQL | 数据库 |

## 核心功能

- **训练计划**: 康复师为患者定制个性化训练方案
- **视频指导**: 训练动作视频演示与语音提示
- **进度跟踪**: 训练完成度统计、康复进度可视化
- **医患互动**: 在线沟通、训练反馈、方案调整

## 主要工作

- 设计训练计划管理系统的前后端架构，采用Vue 3 + Spring Boot + MySQL技术栈
- 实现训练视频播放与进度记录功能，支持视频流媒体播放和进度跟踪
- 开发康复数据可视化报表，直观展示康复进度和训练效果
- 构建医患消息通知系统，实现在线沟通和训练反馈
- 实现完善的权限管理系统，包括角色分离、数据隔离、操作控制和用户认证机制

## 项目成果

- 提升康复训练的规范性与依从性，康复师工作效率显著提高
- 获得康复师和患者的积极反馈，使用体验良好
- 团队技术能力显著提升，积累了全栈开发、权限管理、视频处理等方面的经验
- 项目对康复医疗行业具有实际社会价值，有望在更多机构推广
